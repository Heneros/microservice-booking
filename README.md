Microservice app for booking hotels/apartments, 
with calender (google calendar API) with payment (stripe or paypal)
and notifications(websocket + email).

## Modules
- users
- auth
- booking 
- payment 
- notifications


## Packages
- RabbitMQ
- Prisma
- Redis
